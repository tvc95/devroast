# DevRoast

DevRoast is a web application that lets you roast code and compete on the leaderboard. Built during NLW event from Rocketseat.

## Features

- **Code Input**: Paste or write your code and get instant feedback
- **AI Roasting**: Get humorous and constructive feedback on your code
- **Leaderboard**: Compete with other developers and climb the ranks
- **Dark Mode**: Easy on the eyes, perfect for late-night coding sessions

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Biome

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
